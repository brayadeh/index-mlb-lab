# index-mlb-lab

## 项目简介

`index-mlb-lab` 是一个用于归档和发布独立 HTML 页面的仓库。这里存放的是各类实验性、临时或静态的前端页面，不针对特定域名或网站。每个页面均可独立访问，适合作为功能演示、原型展示或小型工具页面。

## 目录结构

```
index-mlb-lab/
├── index.html          # 仓库首页 / 页面导航
├── page-example/       # 示例页面文件夹
│   ├── index.html
│   └── assets/
└── ...
```

- 根目录下的 `index.html` 可作为页面列表或入口。
- 每个独立页面通常放置在自己的文件夹中，包含 HTML 及相关资源（CSS、JS、图片等）。
- 页面命名应简洁明了，便于识别内容。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无依赖关系。页面内容可能包含：

- 前端技术实验与原型
- 临时展示页
- 静态工具界面
- 学习笔记或代码片段

每个页面均可通过直接访问其 HTML 文件路径来查看。建议在页面内附带简要说明或版本信息。

## 维护说明

- 新增页面：在仓库中创建新文件夹，放入 `index.html` 及相关文件，并在根目录 `index.html` 中添加导航链接（可选）。
- 更新页面：直接修改对应文件夹内的文件，保持版本历史可追溯。
- 删除页面：移除对应文件夹，并清理根目录导航中的链接。
- 请保持页面结构和资源路径的相对性，以便在不同环境下正常访问。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中打开任意 HTML 文件即可查看对应页面。
3. 也可通过 GitHub Pages 或其他静态托管服务发布整个仓库内容。
